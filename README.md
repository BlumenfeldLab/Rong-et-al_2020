# A pulse of fMRI increases in subcortical arousal systems during transitions in attention
This repository contains key codes and files used in "A pulse of fMRI increases in subcortical arousal systems during transitions in attention" - Rong et al, 2020.

## Summary
A detailed description of all methods used in the paper are given in the methods section of the manuscript.  However, to facilitate reproducability of our results, we have included in this repository a number of key codes and files.  If you have further questions regarding this paper, we encourage you to contact [Dr. Hal Blumenfeld](mailto:hal.blumenfeld@yale.edu?subject=[GitHub - Rong et al]
